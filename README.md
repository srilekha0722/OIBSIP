# OIBSIP
# Iris Flower Classification

## Objective

Build a machine learning model to classify Iris flowers into three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Dataset

The Iris dataset contains measurements of sepal length, sepal width, petal length, and petal width.

## Tools and Libraries Used

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

## Algorithm

Random Forest Classifier

## Steps Performed

1. Loaded the Iris dataset.
2. Preprocessed the data.
3. Split the data into training and testing sets.
4. Trained the Random Forest model.
5. Evaluated the model using accuracy score.

## Result

The model successfully classified Iris flowers with high accuracy.
