# Iris Flower Classification

## Objective

Build a machine learning model to classify Iris flowers into three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Dataset

The Iris dataset contains measurements of sepal length, sepal width, petal length, and petal width.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

## Algorithm

Random Forest Classifier

## Steps

1. Load the dataset.
2. Split the data into training and testing sets.
3. Train the Random Forest model.
4. Predict the flower species.
5. Evaluate the model accuracy.

## Result

The model successfully classified Iris flowers with high accuracy.


# Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends in India using Python. The dataset contains information about unemployment rates, employment estimates, labor participation rates, and regional data.

The main objective of this project is to understand unemployment patterns and study the impact of the COVID-19 pandemic on employment across different regions of India.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset Features

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)

## Project Steps

1. Import required libraries.
2. Load the dataset.
3. Explore and clean the data.
4. Check for missing values.
5. Perform data visualization.
6. Analyze unemployment trends.
7. Generate insights from the data.

## Visualizations

* Unemployment Rate Distribution
* Regional Unemployment Analysis
* Correlation Heatmap

## Conclusion

The analysis shows how unemployment rates varied across different regions and highlights the impact of the COVID-19 pandemic on employment conditions in India. Data visualization helped identify patterns and trends within the dataset.
